# TrollGuard - Transformer Based Toxic Comment Detection

## 📌 Description
TrollGuard is a transformer-based web application that detects and filters toxic or abusive comments using advanced Natural Language Processing (NLP) techniques.

## 🚀 Features
- Detects toxic comments in real-time
- Uses transformer-based deep learning model
- User-friendly web interface
- Dashboard for results visualization

## 🛠️ Technologies Used
- Python
- Flask
- Transformer Models (NLP)
- HTML, CSS, JavaScript

## ▶️ How to Run

1. Clone the repository
git clone https://github.com/your-username/trollguard-transformer-toxic-comment-detection.git

2. Navigate to the project folder
cd trollguard-transformer-toxic-comment-detection

3. Install dependencies
pip install -r requirements.txt

4. Run the application
python app.py

5. Open in browser
http://127.0.0.1:5000/

## ⚠️ Note
The trained transformer model is not included due to size limitations.
